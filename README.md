# cs5110-wormy
created by Lexy Simmons and Russell Graves

Worm Movement:
- Green Worm
  - up arrow is up
  - down arrow is down
  - left arrow is left
  - right arrow is right
- Pink Worm
  - W is up
  - S is down
  - A is left
  - D is right
- Both worms
  - keypad 8 is up
  - keypad 2 is down
  - keypad 4 is left
  - keypad 6 is right

Shooting:
- Green Worm shoots with M
- Pink Worm shoots with Q

Additional feature:
- the second worm is pink instead of green
